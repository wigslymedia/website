# Dream Team Automation Website

A modern, metallic-themed website for Dream Team Automation, LLC - Precision Tooling Design.

## Pages

- **Home** (`index.html`) - Hero with animated service keywords and service cards
- **Staffing** (`staffing.html`) - Engineering staffing solutions
- **Services** (`services.html`) - Turn-Key Design, 3D Printing, CAD Automation
- **Projects** (`projects.html`) - Project gallery
- **Careers** (`careers.html`) - Job application form
- **Contact** (`contact.html`) - Quote request form with contact info

## Tech Stack

- Pure HTML5, CSS3, JavaScript (no frameworks)
- Google Fonts (Exo 2, Rajdhani)
- Responsive design
- CSS animations and gradients

## Design

- Dark slate/navy backgrounds (#0f1419, #1a1f2e)
- Metallic silver-to-chrome gradients
- Blue accent highlights (#4a9eff)
- Angular, industrial aesthetic matching the company logo
