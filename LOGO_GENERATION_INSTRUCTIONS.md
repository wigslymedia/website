# Logo Generation Instructions for Nimble Resolve

## Company Overview
**Company Name:** Nimble Resolve  
**Industry:** WiFi Assessment & Network Consulting for Manufacturing Facilities  
**Target Market:** Industrial/manufacturing facilities in Detroit/Michigan area  
**Brand Personality:** Professional, technical, reliable, solution-oriented, industrial expertise

---

## Logo Specifications

### 1. LOGO TYPES TO GENERATE

Generate **3 variations** of the logo:

#### A. Full Logo (Horizontal)
- **Layout:** Company name + icon side-by-side
- **Use Case:** Website header, business cards, letterhead
- **Dimensions:** 180px width × 40px height (or scalable ratio 4.5:1)
- **Orientation:** Horizontal

#### B. Icon-Only Logo (Square)
- **Layout:** Icon only, no text
- **Use Case:** Favicon, social media profile pictures, app icons
- **Dimensions:** 512px × 512px (square)
- **Minimum size:** 32px × 32px (must be readable)

#### C. Stacked Logo (Vertical)
- **Layout:** Icon above company name
- **Use Case:** Mobile headers, square social media posts, vertical spaces
- **Dimensions:** 200px width × 150px height (or scalable ratio 4:3)
- **Orientation:** Vertical

## 3. DESIGN ELEMENTS & CONCEPT

### Icon Concept
The logo should incorporate **WiFi/network signal** imagery combined with **industrial/manufacturing** elements. Consider:

**Primary Icon Ideas:**
- WiFi signal waves (concentric arcs) combined with:
  - Factory/industrial building silhouette
  - Radio frequency waves
  - Network nodes/connection points
  - Signal strength bars
  - Antenna/transmission tower

**Design Style:**
- **Modern & Clean:** Minimalist, professional
- **Technical but Accessible:** Shows expertise without being overly complex
- **Scalable:** Must work at small sizes (favicon) and large sizes (banners)
- **Geometric:** Use clean lines and shapes
- **No gradients in icon:** Solid colors preferred for clarity

### Typography
**Font Style:** 
- Sans-serif, modern, professional
- Similar to: Arial, Helvetica, or similar clean sans-serif
- **Weight:** Bold (700) for company name
- **Letter Spacing:** Slightly tight (-0.02em or similar)

**Text Treatment:**
- "Nimble" - can be slightly lighter weight or same as "Resolve"
- "Resolve" - bold, emphasis on solution-oriented nature
- All caps or Title Case acceptable

---

## 4. DETAILED DESIGN REQUIREMENTS

### Icon Specifications
- **Style:** Flat design, no 3D effects
- **Line Weight:** 2-4px for strokes (scalable)
- **Shape:** Geometric, clean curves
- **Complexity:** Medium - detailed enough to be interesting, simple enough to scale down
- **Color:** Single color or two-color max (primary blue + accent blue)

### Logo Composition
- **Spacing:** Icon and text should have breathing room (20-30% of icon width)
- **Alignment:** Icon and text baseline aligned
- **Proportions:** Icon should be approximately 1:1 square, text height should match icon height

### Technical Requirements
- **Format:** Vector format preferred (SVG), but also provide PNG versions
- **Transparency:** Must support transparent backgrounds
- **Resolution:** 
  - SVG for web (scalable)
  - PNG at 512px, 256px, 128px, 64px, 32px for different use cases
- **File Formats Needed:**
  - SVG (vector, scalable)
  - PNG with transparency (multiple sizes)
  - PDF (for print)
