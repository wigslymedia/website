# Images Directory

Add your optimized images here:

## Recommended Images

1. **Hero Background** (optional)
   - Size: 1920x1080px
   - Format: WebP or optimized JPG
   - File: `hero-bg.jpg` or `hero-bg.webp`

2. **Company Logo**
   - Size: 200x60px (or proportional)
   - Format: PNG with transparency
   - File: `logo.png`

3. **Team Photo** (optional, increases conversions)
   - Size: 800x600px
   - Format: JPG
   - File: `team.jpg`

4. **Process Diagram** (optional)
   - Size: 1200x800px
   - Format: PNG or SVG
   - File: `process-diagram.png`

## Image Optimization

Before adding images:
- Compress using [TinyPNG](https://tinypng.com) or [Squoosh](https://squoosh.app)
- Use WebP format when possible (better compression)
- Keep file sizes under 200KB each
- Use appropriate dimensions (don't upload 4K images for web)

## Usage

Update image paths in `index.html`:
```html
<img src="assets/images/logo.png" alt="Nimble Resolve Logo">
```
